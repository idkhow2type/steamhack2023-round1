## Usage
1. Tải NodeJS, khuyến nghị phiên bản mới nhất (v19.6.0+)
1. Mở terminal trong folder chứa file này
2. Chạy
```bat
npm i
npm run build
```
3. Vào `localhost:3000` và trải nghiệm sản phẩm
